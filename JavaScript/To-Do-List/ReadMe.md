# JavaScript To-Do List App

## 📌 Project Overview

This project is a dynamic To-Do List application built using pure HTML, CSS, and JavaScript.

It allows users to:
- Add tasks
- Delete tasks
- Dynamically update the UI
- Manage application state using arrays

The project demonstrates core frontend development concepts without using any external libraries or frameworks.

---

## 🚀 Features

- Add new tasks
- Remove existing tasks
- Real-time DOM updates
- Clean and structured UI
- State-driven rendering logic

## 🏗️ Architecture Explanation

The application follows a **state-driven rendering approach**:

1. All tasks are stored inside a JavaScript array.
2. The `renderTasks()` function updates the UI.
3. Whenever a task is added or deleted:
   - The array is modified.
   - The UI is completely re-rendered.

This approach reflects how modern frameworks like React manage UI updates conceptually.
